Dotfiles!
=========

Install
-------

 - `git clone git@github.com:sensorario/dotfiles`
 - `cd dotfiles`
 - `./install.sh`

> During installation committer's username and email will be asked and global
> ~/.gitconfig is replaced with desired configuration
